# marvel

> marvel vue js

## Build Setup

``` bash
1 - Download Node.js
Realize o download do node no link: https://nodejs.org/en/

2 - Install Node.js
Instale o node seguindo o setup de instalação

3 - Download Project
Faça o download do projeto e escolha um diretório de sua preferência. Ex: c:/marvel-master

4 - Open Node.js command prompt
Abra o prompt de comando do node.js, vá até o caminho da pasta aonde você extraiu o projeto

5 - Install dependencies
Instale as dependências do projeto: npm install

6 - Serve with hot reload at localhost:8080
Rode o projeto (acesse a url localhost:8080): npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
